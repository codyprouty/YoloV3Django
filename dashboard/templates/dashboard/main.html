Image Detection - count occurrences of objects in photos

This is a website for running YOLO V3 weights and configs. It requires a GPU to host the website, as detections in images require computation. The output is counts in a datasheet format. This is the most useful for our research purposes. The intention was to count honey bees and comb contents of frames, however this website accepts any inputs and can detect any items with the proper files.

Click on add photos and add configs to begin using the system.
